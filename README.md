# Exoplanet Detection — Page-wise Interactive Presentation

This is a static GitHub Pages website for a detailed, slide-by-slide Learners' Space style presentation on **Exoplanet Detection**.

## Features

- Page-wise slides, not a long scrolling article
- Arrow key navigation
- Scroll wheel navigation
- Touch swipe support
- Animated starfield background
- Background animation changes by chapter
- Slide overview / mission map
- Speaker notes panel
- Estimated ~1 hour presentation flow
- No build step, no React, no npm required

## Files

Keep these files in the root of your GitHub repository:

```text
index.html
style.css
script.js
README.md
.gitignore
```

## How to use on GitHub Pages

1. Create a GitHub repository, for example `exoplanet-detection`.
2. Upload all the files from this folder directly into the repository root.
3. Do not upload the zip file itself.
4. Go to `Settings → Pages` inside that repository.
5. Set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Click Save.
7. Your website will be live at:

```text
https://YOUR-USERNAME.github.io/exoplanet-detection/
```

## Controls

- Next slide: `ArrowRight`, `ArrowDown`, `Space`, `PageDown`, scroll down
- Previous slide: `ArrowLeft`, `ArrowUp`, `PageUp`, scroll up
- First slide: `Home`
- Last slide: `End`
- Overview: `O`
- Speaker notes: `N`
- Fullscreen: `F`

## Customization

Edit `script.js` to change slide text, presenter notes, timings, or chapter assignment.
Edit `style.css` to change the visual theme.
